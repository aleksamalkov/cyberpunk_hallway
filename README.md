# Cyberpunk Hallway - Computer Graphics Project

## Keybindings
- Q or F1 - open/close settings and help
- W A S D - move
- ESC - close settings or window

## Implemented Elements
### Basic
- Blending
- Face culling
- Advanced lighting
- Model
### Optional
#### Group A
- Framebuffers
#### Group B
- Normal mapping, Parallax Mapping
- HDR, Bloom
